This folder shows our process for generating blog posts for SEO.

First, you should create a virtual environment from this directory:

```
python -m venv seo
```

Then, you should activate the virtual environment:

```
source seo/bin/activate
```

Next, you need to add an API key to your frontend/.env.local file.

```
ANTHROPIC_API_KEY="sk-ant-yourapi-key"
```
