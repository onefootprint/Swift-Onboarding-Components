---
title: Integrations
apis:
  - method: post
    path: /users/{fp_id}/integrations/alpaca/account
    title: Create an Alpaca account
    description: Forwards a request to open an account at Alpaca for the provided user.
  - method: post
    path: /users/{fp_id}/integrations/alpaca/cip
    title: Forward CIP information to Alpaca
    description: Composes CIP information for the provided user and forwards it to Alpaca.
  - method: post
    path: /users/{fp_id}/integrations/apex/cip_report
    title: Export CIP information for Apex
    description: Builds CIP information for the provided user and returns it as a JSON object.
---

# Integrations

Footprint works seamlessly with your partners. To make sure that you are compliant with your brokerage partners' Customer Identification Programs (CIP), Footprint has first-class support for integrations with a handful of popular parties. If there are parts of your integrations that are missing, reach out - we'd love to make it easier to stay compliant with your partners' CIPs!
