## 1. Overview

This guide explains how to integrate Footprint Onboarding Components into your React applications. If you're not familiar with our KYC flow, we recommend you to check our [guide](/articles/kyc/getting-started).

Onboarding Components take all the magic that resides within the Footprint embedded onboarding and exposes them as individual web/React components you can embed, fully stylize, and control inside your product.

It's a set of unstyled, composable components that can be used to build custom UIs on top of Footprint's APIs.

## 2. Why use Onboarding Components?

Use Onboarding Components if you want more control over the styles and layout of your UI while leveraging Footprint's APIs.
If the [appearance](/articles/integrate/customization) options aren’t enough, Onboarding Components might be what you need.

**Component-first Approach** - Building custom UIs with Footprint is as easy as using our prebuilt components. Onboarding Components manage the business logic for you, so you just need to focus on the UI.

## 3. Browser support

Footprint libraries supports all recent versions of major browsers. For the sake of security and providing the best experience to the majority of customers, we do not support less popular browsers that no longer receive security updates.

| Name                    | Versions              |
| ----------------------- | --------------------- |
| Chrome                  | Last 3 major versions |
| Chrome Mobile           | Last 3 major versions |
| Firefox                 | Last 3 major versions |
| Microsoft Edge          | Last 3 major versions |
| Safari OSX              | Last 3 major versions |
| Safari iOS              | Last 3 major versions |
| Android default browser | Last 3 major versions |

## 4. React support

Footprint requires React 17 or later. We recommend using the latest version of React to take advantage of the latest features and performance improvements.

| Name     | Versions       |
| -------- | -------------- |
| React 18 | Supported      |
| React 17 | Supported      |
| React 16 | Not supported. |

### 5. Installation

To install the Onboarding Components, you need to install the following packages:

```terminal
# With NPM
npm install @onefootprint/footprint-js
npm install @onefootprint/footprint-react@beta

# With yarn
yarn add @onefootprint/footprint-js
yarn add @onefootprint/footprint-react@beta
```
