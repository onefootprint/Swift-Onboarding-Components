# Getting started

1. Go to the Footprint developer dashboard and generate a new Secret API Key. This will look something like sk_test_CJvsN1kaZH3GGtYkaZH3GGtY. Add this to the credential manager of your server-side app. Warning: treat this key like a sensitive password!

2. Check that your API key is working as expected, you should get back some information about this key.
